\problemname{Gluttonous George}
\illustration{0.3}{goggigradugi}{Image taken from \href{https://hdl.handle.net/1946/39680}{Læsisfimman og stærðfræðiþrennan} by Hlín Guðnadóttir. Image is used with permission from Ólöf Kristín Knappet Ásgeirsdóttir.}

Gluttonous George loves eating big numbers.
George is hungry!
George is given two numbers to choose from, but George does not know numbers.
Help George choose the bigger number, otherwise George will be hungry!

In what direction should George's beak face in place of the question mark?

\section*{Input}
Input consists of one line containing first the integer $n$, followed by a space, the question mark \texttt{?}, another space, and finally the integer $m$.

\section*{Output}
If the first number is bigger, then output \textit{>}.
If the second number is bigger, then output \textit{<}.
If the numbers are the same size, then output \textit{Goggi svangur!}.
