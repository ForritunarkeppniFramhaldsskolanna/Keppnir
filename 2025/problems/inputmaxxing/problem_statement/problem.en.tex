\section*{Input}
The first line of input contains a single integer $n$.
Then follow $n$ lines, each containing a single
integer $x_i$ satisfying $10 \leq x_i \leq 10^9$.

\section*{Output}
Print the ``inputmaxx'', that is, the largest value
in the input.
